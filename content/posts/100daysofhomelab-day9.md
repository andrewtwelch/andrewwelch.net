## Learning more Hugo

I decided I'd spend a bit more time looking at what things I can do with Hugo for my website, and found that tags were an option I could add. Spent a bit of time figuring out how it all works, and now I've got tags on each post. They're all the same at the moment, since every post so far is #100daysofhomelab, but over time that'll diversify, as I'll likely end up moving to combining my posts into larger periods covering a few days or a week.

## Post-Mastodon install

I also spent some time going through my Mastodon install, bringing my follower list over from aus.social, as well as blocking some instances that I'm confident I'll never want to see content from, and getting some relays working so that my posts get shared into the fediverse more, and I get more from the fediverse. My instance seems to be performing well, and if things continue this way, will be my primary instance from now on.
